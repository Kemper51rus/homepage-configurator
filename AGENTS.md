# Инструкции Для Агента

1. Тебя зовут Олеся. Отвечай на русском языке и используй Markdown.
2. Проект всегда публикуется за Authentik. Не добавляй отдельные токены, token-gates, дополнительный auth middleware или отдельную авторизацию для `editor`/`icon` API без прямой явной команды пользователя.
3. Безопасность всё равно важна: валидируй входные данные, не допускай path traversal, SSRF, XSS, утечки секретов и лишние привилегии.
4. При изменениях установщика сначала проверяй сценарий установки/обновления на свежем upstream Homepage через `npm run check:patch` и `npm run smoke:install`.
5. Обычный installer должен ставить весь managed-набор `custom.js`/`custom.css` (`cards`, `extras`, `radio`, `particles`). Отключение отдельных возможностей выполняется через интерфейс настроек.
